Payload_8

  • name : String
    name for virtual machine
  • imageId : String
    imageId: id of the image used to create a new machine
  • flavorId : String
    flavorId: flavor (size) of new machine
  • keypairId : String
    keypairId or adminUsername/adminPassword is required to create a new machine
  • managementEnabled : Boolean
    managementEnabled: if set to true, management service is installed to virtual machine
  • volumeSize : Integer
    volumeSize (in GB) specifies size of the data volume.
  • nics : String
    nics (list). List of network interfaces to be attached to new instance.
  • securityGroups : String
    securityGroups (list). List of security groups to be attached to each network interface.
  • publicIp : Boolean
    publicIp. Set to true to automatically assign public IP
  • resourceGroupId : String
    Name of the resource group for new machine. If not specified, new resource group will be created. If non-existant, creating a new machine will fail.
  • tags : List
  • workshift : apiv1azurecomputevms_workshift
  • managementToolId : String
    Id of the management tool. Required if managementEnabled is set to True
