Provider_data

  • name : String
    Name for a provider instance
  • service : String
    \"aws_ses\" or \"k5_smtp\"
  • contractId : String
    Contract id credentaials for K5 SMTP service
  • username : String
    Username credentaials for K5 SMTP service
  • password : String
    Password credentaials for K5 SMTP service
  • accessKey : String
    Access key credentaials for AWS SES service
  • secretKey : String
    Secret access key credentaials for AWS SES service
  • regionId : String
    Region for AWS SES service (optional parameter)
