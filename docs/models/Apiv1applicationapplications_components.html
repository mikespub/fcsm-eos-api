Apiv1applicationapplications_components

  • guid : String
    Component identifier
  • name : String
    Name
  • state : String
    Component deployment state
    Enum:
    queued
    deploying
    active
    error
  • details : String
    Used when state is error. Error details
  • version : String
    Component version to deploy. e.g. 2.1-34
  • size : Integer
    Number of instances to deploy. Each on separate virtual machine.
  • module : String
    Salt module to execute.
  • vms : List
    Virtual machines on which this component is deployed
  • depends_on : String
    In Multi-Component application this fields points to another component on which 'this' components depends on.
