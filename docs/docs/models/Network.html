Network

  • addressPrefix : String
    Address prefix for subnet (i.e., 10.0.0.0/24)
  • networkId : String
    networkId - Network (VPC) ID
  • enableIPv6 : Boolean
    enableIPv6 - if set to true, Amazon will allocate IPv6 range for this network. Defaults to false.
  • instanceTenancy : String
    instanceTenancy - defaults to shared. Set to 'dedicated' to create VPC for dedicated hosts.
