Apiv1applicationblueprints_components

  • name : String
    Template Name
  • module : String
    Salt module to execute.
  • port : Integer
    Component is exposed via this port
  • installation_time : Integer
    Estimated installation time of a component [seconds]
  • size : Integer
    Number of instances to deploy. Each on separate virtual machine.
  • version : String
    Component version to deploy. e.g. 2.1-34
  • instance_aws : apiv1applicationblueprints_instance_aws
  • instance_openstack : apiv1applicationblueprints_instance_aws
  • instance_k5 : apiv1applicationblueprints_instance_aws
  • depends_on : String
    In Multi-Component application this fields points to another component on which 'this' components depends on.
