This is the API

createApplication

Create Application. When 'vms' list is given application's components will be deployed on selected vms. Currenlty only single component Blueprint is supported for vms parameter. Addon application can be created only on standalone virtual machine (not created by application).

URL

https://emeia.fcsm.io/api/v1/application/applications

HTTP Method

POST

Response Type

inline_response_200_8

Parameters

  • Body: application Application(Application)

    Application to create

createBlueprint

Create blueprint

URL

https://emeia.fcsm.io/api/v1/application/blueprints

HTTP Method

POST

Response Type

inline_response_200_7

Parameters

  • Body: blueprint Blueprint(Blueprint)

    Blueprint to create

deleteApplication

Delete application

URL

https://emeia.fcsm.io/api/v1/application/applications/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    Application identifier

  • Query: vms Boolean

    Flag to delete (True) or just unmanage virtual machines (False)

deleteBlueprint

Delete blueprint

URL

HTTP Method

Response Type

Parameters

  • Path: id String

    Blueprint identifier

deleteVmFromApps

Delete vm from application's components. It will only remove VM from database. It will not remove virtual machines from platforms.

URL

https://emeia.fcsm.io/api/v1/application/vms/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    Virtual Machine unique identifier
